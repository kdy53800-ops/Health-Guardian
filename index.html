네이버 아이디로 로그인

Supabase에 네이버 OIDC 공급자가 연결되어 있으면 바로 로그인으로 이동합니다.

로그인 흐름

  • 1.네이버 로그인 한 번으로 가입과 로그인을 동시에 진행합니다.
  • 2.최초 로그인 후 필요한 사용자 정보를 프로필에 자동 연결합니다.
  • 3.로그인 이후 바로 대시보드에서 건강 기록을 시작할 수 있습니다.

관리자 노출 정보

  • •이름, 이메일, 성별, 생일, 휴대전화번호
  • •서비스 내에서 직접 기록한 활동 정보와 목표 설정값
  • •동의한 항목만 저장하며 관리자 화면에도 동일 범위로 표시
