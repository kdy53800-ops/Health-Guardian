건강지킴이

네이버 로그인 기반으로 전환 중인 건강 기록 서비스입니다.

회원가입은 네이버 로그인으로 대체됩니다.

별도의 아이디와 비밀번호를 새로 만드는 대신, 네이버 계정으로 빠르게 시작하고 필요한 최소 정보만 동의받는 구조로 바뀝니다.

네이버 아이디로 시작하기

Supabase에 네이버 OIDC 공급자가 연결되어 있으면 이 버튼이 실제 로그인 진입점으로 동작합니다.

회원가입 흐름 변경

  • 1.사용자는 네이버 로그인 버튼을 통해 가입과 로그인을 한 번에 진행합니다.
  • 2.최초 로그인 후 서비스 이용에 필요한 정보를 자동으로 프로필에 연결합니다.
  • 3.기존 수기 회원가입 폼은 운영 화면에서 제거해도 되는 방향입니다.

관리자에게 제공될 정보

  • •이름, 이메일, 성별, 생일, 휴대전화번호
  • •건강지킴이 서비스에서 직접 수집한 활동 기록과 목표 설정값
  • •네이버 로그인에 동의한 항목만 저장하고, 관리자 화면에도 그 범위만 노출
이름 이메일 성별 생일 휴대전화번호

로컬 계정 삭제

네이버 로그인 전환 이전에 생성된 로컬 테스트 계정을 정리할 때 사용하는 메뉴입니다. 운영 전환 후에는 네이버 연동 기준의 계정 삭제 절차로 바꾸는 것이 자연스럽습니다.

계정을 삭제하면 운동 기록, 목표 설정, 사용자 정보가 함께 제거됩니다. 관리자 계정은 이 메뉴에서 삭제할 수 없습니다.

전환 후 기대 흐름

  • 1 사용자는 네이버 로그인 한 번으로 서비스 가입과 로그인을 동시에 처리합니다.
  • 2 Supabase에는 네이버에서 동의한 프로필 정보와 건강지킴이 활동 데이터가 연결됩니다.
  • 3 관리자는 사용자 랭킹, 사용자 관리, 연락처 확인을 한 화면 흐름 안에서 다룰 수 있습니다.
